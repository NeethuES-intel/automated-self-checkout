    1.5.0

    Intel® Automated Self-Checkout Reference Package 1.5.0 is the second main release. This release includes bug fixes, feature enhancements, dockerization of the benchmarking tools, and OpenVINO Model Server support. For details on running the solution, refer to the Overview.

    New Features

    Title Description
    OpenVINO Model Server OpenVINO Model Server support
    OpenVINO Model Server Pipelines Object detection pipelines using OpenVINO Model Server
    Benchmark scripts Dockerization Benchmark tools have been moved to Docker containers for more flexible deployment
    Github Build actions Code linting and security scans for pull requests
